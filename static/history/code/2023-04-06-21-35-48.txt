以下是找回密码活动图的UML代码和描述：

```plantuml
@startuml
title 找回密码活动图

start
:请求找回密码;
if (验证信息) then (true)
  :发送邮件到绑定邮箱;
  :提示用户;
  :进入邮箱;
  :点击邮件中的链接;
  if (输入新密码) then (true)
    :重置密码;
    :提示密码已重置;
  else (false)
    :提示输入新密码;
    :重新填写新密码;
  endif
else (false)
  :提示验证失败;
  :重新填写验证信息;
endif
stop

@enduml
```

这个活动图描述了用户找回密码的过程。当用户请求找回密码后，系统首先进行验证信息的过程，如果验证成功，则系统将发送重置密码链接到绑定邮箱，用户需要进入邮箱，并点击邮件中的链接。当用户点击链接后，可输入新密码，系统将进行密码的重置。如果用户没有输入新密码，系统将提示用户输入新密码。如果验证信息进行失败，则系统将提示用户重新填写验证信息。

该流程包含了“请求找回密码”、“验证信息”、“发送邮件到绑定邮箱”、“进入邮箱”、“点击邮件中的链接”、“输入新密码”、“重置密码”、“提示密码已重置”、“提示输入新密码”、“提示验证失败”、“重新填写验证信息”等步骤。这个活动图较复杂，但是精确地描述了找回密码的各个步骤。
